# Animieru

## Demo site
http://animieru.chaseterry.com

Uses the Jikan - Unofficial MyAnimeList API ( https://jikan.moe )

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
